# MKOutlinedLabelNode
Draw outlined text in SpriteKit.

Usage
-----
```swift
let textNode = MKOutlinedLabelNode(fontNamed: "Helvetica", fontSize: 32)
textNode.borderColor = UIColor.blackColor()
textNode.fontColor = UIColor.blueColor()
textNode.outlinedText = "Test"
```
